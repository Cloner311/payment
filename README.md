## Requirements

*   PHP >= 7.4
*   PHP ext-curl
*   PHP ext-json
*   PHP ext-soap
*   [Laravel](https://www.laravel.com/) (or [Lumen](https://lumen.laravel.com/)) >= 5.7

---

## **Installation**

Add the package to your composer file via the `composer require` command:

```plaintext
composer require rahabit/payment
```
